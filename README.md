# Conway's Game of Decay

A recursive cellular automaton about false vacuum decay.

Click the universe to nucleate a daughter vacuum. Each bubble mutates the active rules, expands through the grid, collides with other fronts, and can spawn further daughter vacua recursively.

Open `index.html` in a browser or serve the directory with a static file server.

## Deploying on Vercel

Import this repository in Vercel with the default static-site settings:

- Framework preset: Other
- Build command: None
- Output directory: `.`
- Install command: None

The included `vercel.json` enables clean URLs and conservative security/cache headers for the static files.
